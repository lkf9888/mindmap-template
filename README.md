# 思维脑图绘制模板

这是一个纯静态的初版模板，包含节点创建、删除、拖拽、尺寸调整、文字样式编辑、连线编辑、连接线文字注释、画布缩放和平移、多脑图目录，以及只读分享链接。

## 打开方式

直接打开 `index.html`，或在本目录运行：

```bash
python3 -m http.server 5174
```

然后访问 `http://localhost:5174`。

## 当前功能

- 右键画布新建节点，右键节点删除或创建子节点。
- 节点支持标题、文字段落、超链接，以及颜色、形状、文字大小、字体和拖拽尺寸修改。
- 超链接节点会显示“访问 Link”按钮，可直接打开链接。
- 节点颜色提供 24 个快速色块，也保留自定义颜色输入。
- 右键节点支持复制本节点，节点可拖到任意方向，不限制在固定画布边界内。
- 连接线支持颜色、直线/曲线/折线、箭头方向和文字注释。
- 鼠标滚轮缩放，拖动画布平移，工具栏可横向/纵向整理节点。
- 左侧目录支持创建多个脑图、切换脑图、重命名当前脑图，数据保存在浏览器本地存储中。
- 顶部语言按钮支持简体中文/英文界面切换。
- 点击“分享只读链接”会复制一个带脑图快照的 URL，别人打开后只能查看、缩放和平移。
